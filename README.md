# 📌 About The Project

DesignHub is a modern and visually premium frontend website inspired by platforms like Dribbble and modern SaaS landing pages. The project focuses on clean UI, smooth animations, responsive layouts, and modern web design trends.

The goal of this project was to practice:

* Responsive Web Design
* Modern UI/UX Principles
* CSS Animations
* Layout Structuring
* Interactive Components
* Premium Landing Page Design

# ✨ Features

* Modern Dark UI
